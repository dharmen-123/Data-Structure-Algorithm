
    // // ******* ARRAY ******* // //

    An array is a collection of elements of the same data type, 
    stored in contiguous memory locations, and accessed using an index.

    In simple terms:
Array = ordered collection + same type + continuous memory

    // Characteristics of Array //
 ~~> Stores homogeneous elements (same data type)
 ~~> Elements are stored in contiguous memory
 ~~> Each element is accessed using an index
 ~~> Index starts from 0 (in C++)
 ~~> Fixed size (in static arrays)

🔹 Declaration

int arr[5];

🔹 Initialization

int arr[5] = {10, 20, 30, 40, 50};

🔹 Accessing Elements
cout << arr[0];  // 10
cout << arr[2];  // 30

🔹 Memory Representation
Example:
int arr[5] = {10, 20, 30, 40, 50};

Memory layout:

Index  Value Address
 0	   10	 1000
 1	   20	 1004
 2	   30	 1008
 3	   40	 1012
 4	   50	 1016

Address formula:
    Address = Base Address+(Index * Size of element)
